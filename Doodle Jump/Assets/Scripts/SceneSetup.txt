Настройка многосценарной архитектуры (Zenject + MVVM)

=== СТРУКТУРА СЦЕН ===

1. BootstrapScene — подготовка к запуску, переход в MenuScene
2. MenuScene — главное меню с кнопками Play и Exit
3. GameplayScene — игровой процесс (геймплей)

=== ПОРЯДОК ВЫПОЛНЕНИЯ ===

A. ProjectContext (Resources/ProjectContext.prefab)
   - Уже настроен с ProjectInstaller
   - Содержит: GameConfig, SignalBus, Signals, SceneLoadingService
   - ProjectContext загружается автоматически при старте первой сцены

B. Создание сцен

   1. BootstrapScene:
      - Создай новую сцену, сохрани как BootstrapScene
      - Создай пустой GameObject "SceneContext"
      - Добавь компонент SceneContext (Zenject)
      - Создай пустой GameObject "BootstrapInstaller", добавь скрипт BootstrapInstaller
      - В SceneContext → Installers добавь BootstrapInstaller
      - Сцена может быть пустой (без камеры — Zenject создаст при необходимости)

   2. MenuScene:
      - Создай новую сцену, сохрани как MenuScene
      - Добавь Camera (Main Camera)
      - Создай Canvas (UI) с кнопками Play и Exit
      - Создай пустой GameObject "MenuView", добавь компонент MenuView
      - Привяжи кнопки в инспекторе: Play Button → _playButton, Exit Button → _exitButton
      - Создай GameObject "SceneContext", добавь SceneContext
      - Создай GameObject "MenuInstaller", добавь скрипт MenuInstaller
      - В MenuInstaller назначь MenuView (или оставь пустым — найдётся автоматически)
      - В SceneContext → Installers добавь MenuInstaller

   3. GameplayScene:
      - Переименуй SampleScene в GameplayScene (или скопируй содержимое)
      - Создай GameObject "SceneContext", добавь SceneContext
      - Создай GameObject "GameInstaller", добавь скрипт GameInstaller
      - Создай/найди GameObject с InputInstaller, PlayerInstaller, PlatformsInstaller, UiInstaller
      - В SceneContext → Installers добавь по порядку:
        - GameInstaller
        - InputInstaller
        - PlayerInstaller (укажи PlayerConfig, PlayerView)
        - PlatformsInstaller (укажи Platform prefab)
        - UiInstaller (укажи GameHudView)
      - В GameHudView (панель Game Over) добавь две кнопки:
        - "Начать заново" → привяжи к _restartButton
        - "Выйти в меню" → привяжи к _exitToMenuButton

C. Build Settings
   - File → Build Settings
   - Добавь сцены в порядке: BootstrapScene (0), MenuScene (1), GameplayScene (2)
   - BootstrapScene должна быть первой (индекс 0)

D. GameConfig и PlayerConfig
   - GameConfig: назначается в ProjectInstaller (ProjectContext)
   - PlayerConfig: назначается в PlayerInstaller (SceneContext GameplayScene)

E. Дополнительно для GameplayScene
   - Main Camera: добавь CameraFollower (опционально)
   - Prefab платформы: Collider2D, PlatformView (Bounce Force)
   - HUD: Score Text, Game Over Panel (скрыта по умолчанию), кнопки Restart и Exit To Menu
